Train TTS
===========
该项目实现了从串口实时读取数据，并存入本地SQLite数据库，同时满足一定条件触发TTS响应的简单流程。

Install
-----------
需要先安装preinstall目录下面的两个apk

Features
-----------
* 实现串口数据的读取
* 实现SQLite数据接口
* 实现Android TTS的调用
